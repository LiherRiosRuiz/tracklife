# Memoria de Platón

Índice de sesiones y entidades guardadas.
Este archivo es generado automáticamente por Platón.

## Cómo usar

- `/mem list` — listar todo
- `/mem save <título>` — guardar última respuesta como entidad
- `/mem search <consulta>` — buscar en toda la memoria
- `/end` — guardar resumen de la sesión y salir

## Estructura

```
.sdd/memory/
  sessions/   ← un .md por sesión: YYYY-MM-DD-HHmm.md
  entities/   ← memorias nombradas: título-en-slug.md
```
