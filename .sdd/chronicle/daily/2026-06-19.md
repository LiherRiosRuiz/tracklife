### 5. Fix quevedo.mjs â€” enlaces rotos por code spans

**Problema**: Quevedo mostraba 2 enlaces rotos (`[[Nota\|Alias]]`).

**Causa**: `buildVaultGraph` parseaba wikilinks sobre el contenido raw,
incluyendo ejemplos de sintaxis dentro de code spans en `docs/Cronica.md`
y `docs/Quevedo.md`. El string `[[Nota\|Alias]]` escrito como ejemplo
se trataba como enlace real â†’ destino `Nota\` no existia.

**Fix** en `.sdd/cli/quevedo.mjs`, funcion `buildVaultGraph`:
strip de bloques de codigo e inline code spans antes del regex de wikilinks:
```javascript
const content = raw
  .replace(/```[\s\S]*?```/g, "")
  .replace(/`[^`\n]+`/g, "");
```

**Resultado**: 0 rotos, 0 huerfanas en el vault.
